Extension pairing

Pair this browser to one saved Firestore policy document. The pairing code is the recommended PoC path because it bundles the public Firebase settings the extension needs for polling.

Recommended setup

The pairing code carries the policy ID plus the public Firebase project settings required for remote polling in this PoC.

Manual fallback

Manual entry is only here to satisfy raw policy-ID pairing in the PoC. These are public web config values, not secret admin credentials.

This Phase 4 pairing model is intentionally narrow. It does not implement device identity, hardened enrollment, or kiosk protections.
